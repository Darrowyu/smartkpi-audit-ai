# 考核周期

考核周期定义了绩效评估的时间窗口。本章介绍如何创建和管理考核周期。

## 进入考核周期管理

入口：**左侧导航 → 考核周期**

页面显示所有考核周期列表，包括周期名称、起止日期和状态。

## 创建考核周期

1. 点击「新建周期」按钮
2. 填写周期信息：
   - **周期名称** - 如"2024年第一季度考核"
   - **开始日期** - 考核起始时间
   - **结束日期** - 考核截止时间

3. 点击「创建」完成

## 周期状态

| 状态 | 说明 |
|------|------|
| **DRAFT** | 草稿，可编辑和删除 |
| **ACTIVE** | 进行中，可录入数据 |
| **LOCKED** | 已锁定，数据不可修改 |

## 锁定周期

锁定操作用于冻结周期数据：

1. 在周期列表找到目标周期
2. 点击「锁定」按钮
3. 确认锁定

> **警告**：锁定后将无法修改指标分配和数据提交，请确认数据完整后再锁定。

## 指标分配

创建周期后需要分配考核指标。

入口：**左侧导航 → 指标分配**

### 操作步骤

1. 选择目标考核周期
2. 选择部门（可选全部部门）
3. 点击「新增分配」
4. 选择 KPI 指标
5. 填写目标值、挑战值和权重
6. 保存分配

### 权重说明

- 所有指标权重之和应为 100%
- 页面会显示当前权重合计
- 权重不足或超出会有提示

## 复制分配

可以将一个周期的指标分配复制到另一个周期：

1. 在指标分配页面
2. 选择要复制的周期
3. 使用复制功能快速创建新周期分配

## 注意事项

1. **时间规划** - 预留充足的数据填报时间
2. **指标权重** - 确保权重合计为 100%
3. **锁定谨慎** - 锁定前确认数据完整
4. **周期锁定后** - 无法新增或修改指标分配
