# 数据填报

数据填报是将 KPI 实际完成值录入系统的过程。根据角色不同，有两种填报方式。

## 经理/管理员填报

入口：**左侧导航 → 数据填报**

### 手动填报

1. **选择考核周期** - 从下拉菜单选择目标周期
2. **切换到「手动填报」标签**
3. **添加条目** - 点击「添加条目」按钮
4. **选择员工和指标** - 从下拉框选择
5. **填写实际值和备注**
6. **保存或提交** - 点击「保存草稿」暂存，点击「提交」正式提交

### Excel 批量导入

1. 点击「下载模板」获取当前周期的 Excel 模板
2. 按模板格式填写各员工的指标数据
3. 点击「上传文件」选择填好的 Excel
4. 系统校验并导入数据

> **提示**：模板会根据当前周期的指标分配自动生成对应列。

## 员工自评填报

入口：**左侧导航 → 自评中心**

普通员工填报自己的 KPI 实际值：

1. 进入自评中心页面
2. 查看分配给自己的 KPI 指标列表
3. 在「实际值」列填写完成情况
4. 可选填「备注说明」解释特殊情况
5. 点击「保存草稿」暂存本地
6. 确认无误后点击「提交自评」

### 填报提示

- 请如实填写各项 KPI 的实际完成值
- 可以添加备注说明特殊情况
- 草稿会自动保存在本地浏览器
- 提交后将进入审批流程，审批通过后计入最终得分

## 数据状态

| 状态 | 说明 |
|------|------|
| **草稿** | 已保存但未提交，可修改 |
| **待审批** | 已提交，等待上级审批 |
| **已通过** | 审批通过，数据生效 |
| **已驳回** | 审批未通过，需修改重提 |

## 数据审批

提交的数据需要经过审批：

入口：**左侧导航 → 数据审批**（仅经理/管理员可见）

1. 查看待审批的数据提交列表
2. 审核数据的合理性
3. 点击「通过」或「驳回」
4. 驳回时需说明原因

## 注意事项

1. **截止日期** - 注意考核周期的数据提交截止时间
2. **数据准确** - 提交前仔细核对数值
3. **及时沟通** - 数据异常及时与上级沟通
4. **本周期限制** - 每个周期只能提交一次自评
