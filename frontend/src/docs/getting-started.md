# 快速入门

欢迎使用 **Makrite KPI 绩效管理系统**！本指南将帮助您快速了解系统的基本功能和操作方式。

## 系统简介

Makrite KPI 是一套企业级绩效管理解决方案，主要功能模块：

**业务功能（经理/管理员）**
- **仪表盘** - 实时监控团队绩效状态与结果
- **指标库** - 创建和维护 KPI 指标定义
- **考核周期** - 管理绩效考核的时间窗口
- **指标分配** - 为部门/员工分配 KPI 目标
- **数据填报** - 录入 KPI 实际完成值
- **数据审批** - 审核提交的绩效数据
- **进度检查** - 跟踪 KPI 完成进度
- **绩效面谈** - 记录绩效沟通内容
- **报表中心** - 查看绩效报表和排名

**个人功能（普通员工）**
- **个人仪表盘** - 查看个人绩效概览
- **我的 KPI** - 查看分配的指标详情
- **自评中心** - 填报实际完成值

## 首次登录

1. 打开系统登录页面
2. 输入管理员分配的**用户名**和**初始密码**
3. 点击「立即登录」按钮
4. 首次登录建议立即在个人设置中修改密码

> **提示**：如果忘记密码，可点击登录页的「忘记密码?」链接，通过邮箱重置。

## 界面导览

登录后您将看到主界面，包含以下区域：

| 区域 | 说明 |
|------|------|
| **左侧导航栏** | 功能菜单入口，可折叠收起 |
| **顶部工具栏** | 语言切换、深色/浅色主题、通知中心 |
| **主内容区** | 当前功能的操作界面 |

## 用户角色

系统支持四种用户角色，权限由高到低：

1. **超级管理员** (SUPER_ADMIN) - 平台全局管理，可管理所有集团和公司
2. **集团管理员** (GROUP_ADMIN) - 管理集团内所有子公司的数据
3. **经理** (MANAGER) - 管理本部门的考核数据，可审批下属数据
4. **普通用户** (USER) - 仅可查看和填报个人 KPI 数据
