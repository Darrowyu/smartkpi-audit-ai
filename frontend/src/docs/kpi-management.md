# KPI 指标库

KPI（关键绩效指标）是衡量员工和团队绩效的核心工具。本章介绍如何创建和管理 KPI 指标定义。

## 进入指标库

入口：**左侧导航 → 指标库**

指标库页面显示所有已创建的 KPI 指标，支持搜索和筛选。

## 创建新指标

1. 点击页面右上角「创建指标」按钮
2. 填写指标信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| **指标编码** | 唯一标识符 | SALES_RATE |
| **指标名称** | 指标显示名称 | 销售额完成率 |
| **描述** | 指标详细说明 | 实际销售额与目标的比值 |
| **计算公式** | 得分计算方式 | 越高越好 |
| **考核频率** | 评估周期 | 季度 |
| **默认权重** | 建议权重占比 | 20% |
| **单位** | 数值单位 | % |
| **得分上限** | 最高得分 | 120 |
| **得分下限** | 最低得分 | 0 |

3. 点击「创建」完成

## 计算公式类型

系统支持五种计算公式：

| 公式类型 | 说明 | 适用场景 |
|---------|------|---------|
| **越高越好** | 实际值越高得分越高 | 销售额、产量 |
| **越低越好** | 实际值越低得分越高 | 成本、投诉率 |
| **区间达标** | 在目标区间内得满分 | 库存周转率 |
| **是否达标** | 达成100分，未达成0分 | 合规检查 |
| **自定义** | 自定义计算公式 | 复杂计算逻辑 |

## 编辑和删除

- **编辑**：点击操作列的编辑图标，修改信息后保存
- **删除**：点击删除图标，确认后删除

> **注意**：已被指标分配引用的 KPI 删除需谨慎。

## 最佳实践

1. **编码规范** - 使用统一的编码格式，如 `DEPT_指标类型_序号`
2. **合理权重** - 默认权重仅供参考，实际分配时可调整
3. **清晰描述** - 描述中说明计算口径和数据来源
4. **定期复核** - 每季度审视指标有效性和合理性
