# 权限管理

权限管理用于配置基于角色的访问控制。本章介绍系统的权限体系和配置方法。

## 进入权限管理

入口：**左侧导航 → 组织管理 → 权限管理**

> **权限要求**：仅管理员可访问

## 角色体系

系统预设四种角色：

| 角色 | 英文代码 | 权限范围 |
|------|---------|---------|
| 超级管理员 | SUPER_ADMIN | 系统全部功能 |
| 集团管理员 | GROUP_ADMIN | 集团内所有公司 |
| 经理 | MANAGER | 本部门管理权限 |
| 普通用户 | USER | 个人数据查看 |

## 权限配置

在权限管理页面可以查看和配置各角色的权限：

1. 从下拉菜单选择角色
2. 查看该角色的权限列表
3. 勾选或取消勾选权限项
4. 点击「保存更改」生效

## 权限模块

系统权限按模块划分：

| 模块 | 权限项 |
|------|--------|
| **KPI 指标库** | 查看、创建、编辑、删除指标 |
| **考核周期** | 查看、创建周期、锁定周期 |
| **数据填报** | 查看、提交、审批填报数据 |
| **报表中心** | 查看报表、导出报表 |
| **用户管理** | 查看、创建、编辑、删除用户 |
| **系统设置** | 查看设置、修改设置 |

## 各角色默认权限

### 超级管理员
- 所有模块的全部权限

### 集团管理员
- KPI 库：查看、创建、编辑、删除
- 考核周期：查看、创建、锁定
- 数据填报：查看、提交、审批
- 报表：查看、导出
- 用户管理：查看、创建、编辑、删除
- 系统设置：查看、修改

### 经理
- KPI 库：查看
- 考核周期：查看
- 数据填报：查看、提交、审批
- 报表：查看
- 用户管理：查看

### 普通用户
- KPI 库：查看
- 数据填报：查看、提交
- 报表：查看（仅个人数据）

## 重置权限

如果权限配置出错，可点击「重置」恢复默认设置。

## 常见问题

**Q: 为什么看不到某个菜单？**
A: 您的角色可能没有该功能的访问权限，请联系管理员。

**Q: 权限变更后多久生效？**
A: 重新登录后立即生效。
