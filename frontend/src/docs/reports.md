# 报表中心

报表中心提供多维度的绩效分析和数据可视化，帮助管理者了解团队和个人的绩效表现。

## 进入报表

入口：**左侧导航 → 报表中心**

## 绩效概览

页面顶部显示选定考核周期的整体指标：

| 指标 | 说明 |
|------|------|
| **总参评人数** | 参与本周期考核的员工数 |
| **平均得分** | 全员平均绩效分数 |
| **优秀率** | 评级为优秀的员工占比 |
| **待改进** | 需制定改进计划的人数 |

## 部门绩效排名

左侧展示部门排名列表：

- 按平均得分从高到低排序
- 显示部门名称、均分、人数
- 点击部门可查看该部门员工明细

## 员工绩效排名

右侧展示员工排名表格：

| 列 | 说明 |
|----|------|
| 排名 | 在公司内的排名 |
| 姓名 | 员工姓名 |
| 部门 | 所属部门 |
| 得分 | 综合绩效得分 |
| 等级 | 绩效等级（优秀/良好/一般/待改进） |

支持功能：
- 按部门筛选
- 点击员工查看指标明细

## 绩效趋势分析

展示绩效的时间变化趋势：

- **折线图** - 最近 6 个考核周期的平均分变化
- **参评人数** - 各周期参与考核的人数
- **对比线** - 及格线和优秀线参考

## 低绩效预警

列出需要关注的低绩效员工：

- 得分低于设定阈值的员工
- 连续多周期表现不佳的员工
- 便于及时制定改进计划

## 导出功能

支持将报表导出为 Excel 文件：

1. 点击「导出报表」按钮
2. 选择导出类型（员工/部门/明细）
3. 下载生成的 Excel 文件

## 重新计算

如果数据有更新，可手动触发重新计算：

1. 点击「重新计算」按钮
2. 系统重新计算所有员工得分
3. 报表数据自动刷新

## 数据权限

不同角色看到的数据范围不同：

| 角色 | 可见范围 |
|------|---------|
| 超级管理员 | 全部数据 |
| 集团管理员 | 集团内所有公司 |
| 经理 | 本部门数据 |
| 普通用户 | 仅个人数据 |
