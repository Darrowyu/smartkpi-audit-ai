# 系统设置

系统设置允许用户个性化配置界面和功能偏好。本章介绍各项设置的使用方法。

## 进入设置

入口：点击左下角头像 → 选择「个人设置」

设置页面包含多个标签页：个人资料、外观、通知、安全。

## 个人资料

管理您的账户基本信息：

### 可修改项目
- **头像** - 点击上传个人照片
- **姓/名** - 您的真实姓名
- **邮箱** - 联系邮箱

### 不可修改项目
- 用户名（登录账号）
- 角色
- 所属部门

> 如需修改角色或部门，请联系管理员。

## 外观设置

自定义界面显示效果：

### 主题模式
- **浅色模式** - 白色背景，适合日间使用
- **深色模式** - 深色背景，减少眼睛疲劳
- **跟随系统** - 自动跟随操作系统设置

### 主题色
可选择不同的强调色：
- 蓝色（默认）
- 青色
- 紫色
- 橙色
- 自定义颜色

> 主题色会影响侧边栏和按钮的颜色。

## 通知设置

配置系统通知偏好：

### 通知类型
| 类型 | 说明 |
|------|------|
| 邮件通知 | 重要事项发送邮件 |
| 系统通知 | 站内消息提醒 |
| KPI 提醒 | 考核相关提醒 |
| 审批提醒 | 审批状态变更提醒 |
| 周报摘要 | 每周绩效摘要 |

### 设置方式
1. 进入「通知」标签
2. 开启或关闭各类通知
3. 修改自动保存

## 安全设置

管理账户安全：

### 修改密码

1. 进入「安全」标签
2. 输入当前密码
3. 输入新密码（至少 6 位）
4. 确认新密码
5. 点击「更新密码」

### 登录历史

查看最近的登录记录：
- 登录时间
- IP 地址
- 登录位置
- 设备信息

> 如发现异常登录，请立即修改密码。

### 会话管理

- 查看当前活跃会话
- 可撤销其他设备的登录状态

## 语言设置

顶部工具栏可快速切换界面语言：
- 简体中文
- English

点击即时切换，无需刷新页面。

## 注意事项

1. **密码安全** - 建议定期更换密码
2. **隐私保护** - 不要在公共设备保存登录状态
3. **异常处理** - 发现异常登录请立即修改密码并联系管理员
