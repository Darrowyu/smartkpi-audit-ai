# 团队管理

团队管理功能用于维护用户账户和组织信息。本章介绍用户管理和组织设置操作。

## 用户管理

入口：**左侧导航 → 组织管理 → 用户管理**

> **权限要求**：仅管理员可访问

### 用户列表

页面显示所有用户，包括：
- 用户名和姓名
- 邮箱地址
- 所属公司和部门
- 用户角色
- 最后登录时间

支持按角色筛选和搜索用户名。

### 创建用户

1. 点击「添加用户」按钮
2. 填写用户信息：

| 字段 | 说明 |
|------|------|
| **用户名** | 登录账号，至少 2 个字符 |
| **邮箱** | 用于登录和通知 |
| **密码** | 初始密码，至少 6 个字符 |
| **姓/名** | 用户真实姓名 |
| **角色** | 选择用户角色 |
| **公司** | 所属子公司 |
| **部门** | 所属部门（可选） |

3. 点击「创建」完成

### 编辑用户

1. 在用户列表点击编辑图标
2. 修改用户信息
3. 密码留空则保持不变
4. 点击「更新」保存

### 删除用户

1. 点击删除图标
2. 确认删除操作

> **注意**：删除用户不可恢复，历史数据会保留。

## 组织设置

### 集团设置

入口：**组织管理 → 集团设置**

管理集团基本信息：
- 集团名称
- 集团代码
- 域名标识

### 公司设置

入口：**组织管理 → 公司设置**

管理子公司信息：
- 添加/编辑/删除子公司
- 设置公司名称和代码

### 部门管理

在公司设置页面管理部门：
- 添加/编辑/删除部门
- 设置部门名称、代码和描述

## 数据安全

1. **权限控制** - 仅管理员可管理用户
2. **操作确认** - 删除操作需二次确认
3. **密码安全** - 建议定期修改密码
